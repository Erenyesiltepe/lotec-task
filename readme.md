## Setup guide
### Server
1) Create a terminal in server folder
2) run pnpm i
3) run pnpm prisma:generate
4) run pnpm prisma:push
5) run pnpm dev

### Client
1) Create a terminal in the client folder
2) run pnpm i
3) run pnpm dev

Note: You may test the api seperately by using the postman.json file in the root folder.
